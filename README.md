# Lista della Spesa

App web per gestire la lista della spesa con categorie e persistenza locale.

## Stack

| Tecnologia | Uso |
|------------|-----|
| React 18 | UI |
| Vite | Build |
| Tailwind CSS | Styling |
| localStorage | Persistenza |
| Vercel | Deploy |

## Funzionalità

- Aggiungi prodotti con nome, quantità e categoria
- Categorie: Frutta/Verdura, Latticini, Carne/Pesce, Surgelati, Dispensa, Bevande, Igiene, Casa, Altro
- Spunta prodotti acquistati
- Elimina prodotti singoli o tutti i completati
- Dati salvati nel browser

## Setup Locale

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
vercel --prod
```
